System Overview

Kloqo is a comprehensive healthcare appointment management system built as a monorepo with multiple Next.js applications. The system manages appointments, walk-ins, patient records, doctor schedules, and real-time queue management for healthcare clinics.

Key Features

  • Multi-App Architecture: Separate applications for patients, nurses, clinic admins, and superadmins
  • Advanced Booking: Patients can book appointments in advance with 85/15 capacity split
  • Walk-in Management: Real-time walk-in scheduling with intelligent slot assignment
  • Token System: A-tokens (advanced) and W-tokens (walk-in) with automatic generation
  • Status Management: Automatic status transitions (Pending ‚Üí Skipped ‚Üí No-show)
  • Queue Management: Real-time consultation queue with position tracking
  • Notifications: Push notifications for appointment updates
  • Break Management: Doctor break periods with schedule adjustments
  • Multi-language Support: English and Malayalam support
