Application Purpose

Kloqo is a comprehensive healthcare appointment management system designed to streamline clinic operations, patient scheduling, and queue management. The system serves healthcare clinics by providing:

  • Multi-channel Booking: Patients can book appointments in advance or walk in for immediate consultation
  • Real-time Queue Management: Live tracking of consultation queues with position updates
  • Intelligent Scheduling: Automatic slot assignment with capacity management (85% advance, 15% walk-in)
  • Status Automation: Automatic status transitions based on time and doctor availability
  • Multi-role Support: Separate interfaces for patients, nurses, clinic administrators, and superadmins
  • Notification System: Real-time push notifications for appointment updates

Business Domain

Kloqo operates in the healthcare appointment management domain, specifically targeting:

  • Healthcare Clinics: Private clinics, hospitals, and medical facilities
  • Multi-doctor Practices: Clinics with multiple doctors and departments
  • Walk-in & Appointment-based Services: Facilities that handle both scheduled and walk-in patients
  • Token-based Queue Systems: Clinics using token numbers for patient management

Key Features & Capabilities

1. Appointment Management

  • Advanced Booking: Patients can book appointments days in advance
  • Walk-in Booking: Real-time walk-in scheduling with intelligent slot placement
  • Token System: A-tokens (advanced) and W-tokens (walk-in) with automatic generation
  • Slot Reservation: Atomic slot locking to prevent double booking
  • Status Management: Pending â†’ Confirmed â†’ Skipped â†’ No-show â†’ Completed

2. Queue Management

  • Real-time Queue: Live consultation queue with position tracking
  • Buffer Queue: Top 2 patients ready for consultation
  • Skipped Token Rejoin: Automatic reinsertion of skipped tokens
  • Consultation Counter: Track completed consultations per doctor/session

3. Doctor & Schedule Management

  • Availability Slots: Day-wise availability with multiple sessions
  • Break Management: Doctor break periods with automatic schedule adjustment
  • Availability Extensions: Extend sessions beyond normal hours
  • Leave Management: Block specific dates/slots for doctor leaves

4. Patient Management

  • Patient Profiles: Complete patient information with visit history
  • Relative Management: Link family members to primary patient
  • Multi-clinic Support: Patients can visit multiple clinics
  • Kloqo Membership: Track registered vs non-registered patients

5. Notifications

  • Push Notifications: Real-time updates via Firebase Cloud Messaging
  • SMS Integration: Twilio integration for SMS notifications
  • Notification Types: Booking confirmations, token calls, break updates, appointment cancellations

Quick Start Guide

Prerequisites

  • Node.js: >= 18.0.0
  • pnpm: >= 8.0.0
  • Firebase Account: With project configured

Installation

# Install pnpm globally (if not installed)
npm install -g pnpm

# Clone the repository
git clone [repository-url]
cd Kloqo-Production

# Install dependencies
pnpm install

# Set up environment variables
# Copy .env.example to .env.local in each app directory
# apps/patient-app/.env.local
# apps/nurse-app/.env.local
# apps/clinic-admin/.env.local

Environment Variables

Each app requires Firebase configuration:

NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id

Running the Applications

# Run all apps in development mode
pnpm dev

# Or run specific apps
pnpm dev:patient      # Patient app (http://localhost:3000)
pnpm dev:nurse        # Nurse app (http://localhost:3001)
pnpm dev:clinic       # Clinic admin (http://localhost:3002)
pnpm dev:superadmin   # Superadmin (http://localhost:3003)

Building for Production

# Build all apps
pnpm build

# Type check
pnpm typecheck

# Run tests
pnpm test
