Routine Tasks

  • Review Firestore rules quarterly.
  • Rotate service account keys; refresh env vars.
  • Clean old slot-reservations if any stale docs remain (should be transactional).

Database

  • Export Firestore snapshots before major releases.
  • Monitor appointment/doc counts; add composite indexes as queries grow.

Dependencies

  • Run `pnpm outdated`; update minor versions monthly.
  • Retest core flows after updates: booking, walk-in, status updates.

Health Checks

  • Add `/api/health` endpoints (not present) for uptime monitors.
  • Check FCM token registration rates; alert if dropping.
